
# Інструкція користувача DigiUni Registry

При запуску програми користувач потрапляє до меню авторизації, де для входу в систему необхідно ввести логін та пароль.
Після успішної перевірки даних обсяг доступних функцій визначається автоматично згідно з призначеною роллю:
- Роль «Користувач» передбачає роботу лише в режимі перегляду, що включає доступ до звітів та функцій пошуку.
- «Менеджер» має повний доступ до інструментів створення, редагування та видалення записів про підрозділи університету, студентів і викладачів, проте не може керувати обліковими записами інших користувачів.
- Найвищий рівень доступу має «Адміністратор», якому відкриті всі опції системи.

Взаємодія із системою реалізована через інтуїтивно зрозуміле консольне меню, де навігація здійснюється шляхом введення відповідних чисел згідно з підказками на екрані. Після обрання необхідного режиму програма надає інструкції щодо подальших дій.
Структура інтерфейсу дозволяє користувачеві після виконання кожної задачі повертатися до попередніх рівнів меню або завершувати робочу сесію за потреби.
